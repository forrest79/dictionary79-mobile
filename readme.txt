English - Czech dictionary for mobile phones (J2ME).


REQUIREMENTS
============
Mobile device compatible with J2ME.


SOURCE CODE
===========
First, use PrepareDictionary. Input file is text file with translated words separate with tabs and with one translate on one line.

Example:
car	auto
bus	autobus

Create "data" subfolder and then copy all *.index and *.dat files to J2ME /data directory.
